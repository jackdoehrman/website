Security Entry ROI & Optimization

Operational Analysis by Jack Doehrman

This analysis evaluated the return on investment for enhanced security systems by quantifying time savings, staffing reductions, and revenue impact across event operations.

Overview

The goal of this project was to evaluate the possible revenue we could gain by upgrading our security screening systems and enabling quicker entry times for fans. This monetization analysis evaluated whether the merchandise and concession sales caused by quicker entry times would help add to the ROI case to upgrade these systems.

Data Sources

  • Event attendance and gate entry timestamps
  • Net revenue for merchanidse and concession sales
  • Hourly staffing schedules and labor costs
  • Security vendor pricing and upgrade options

Analysis Approach

I modeled this analysis around equating the time waiting in line to Lost Revenue. I started out with the ticket scan timestamps and calculating the "Time until Kick" for each scan. This allowed me to bucket and visualize the traffic for each gate. Using a Polynomial trendline, I defined the "Maximum Capacity" times, which 5-minute intervals were working above the average "People per Minute".
